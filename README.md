## About The Project:
Webapp to aid in early color/theming stages of UI design process.

Uses local storage to save updates to color palettes.

<!-- GETTING STARTED -->
## Getting Started

To get a local copy up and running follow these simple example steps.
<br>
<br>
### Prerequisites:

    Node version >= 14

### Installation:

1. Clone the repo
   ```sh
   git clone https://github.com/itBme3/coloring-palettes
   ```
2. Install packages
   ```sh
   yarn install
   ```
3. Generate content
   ```sh
   yarn run generate
   ```
4. Serve Locally
      ```sh
      yarn run start
      ```
   Or Development (hot reloading using Vite)
      ```sh
      yarn run dev
      ```
